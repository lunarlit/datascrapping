위 쇼핑몰 HTML 구조를 분석하고, 다음에서 요구하는 요소의 단일 선택자 혹은 선택자 경로를 찾아보세요.

1. 품명에 해당하는 HTML 요소
2. 물품 이미지에 해당하는 HTML 요소
3. 물품 정보 HTML 요소의 자식들 중 상품평에 해당하는 HTML 요소
4. 전체 물품 리스트 HTML 요소의 자손들 중 품명, 가격, 상품평 등의 데이터를 가진 HTML 요소들
